# Matteo-CLI-MusicTool
A CLI music player written in Python
This is entirely written by me, but originally, inspiration was taken from a YouTube video. 

All you have to do is unzip the file, and run the exe inside! When prompted, enter a number to make your choice.

The Python source code requires the libraries "colorama" and "pygame" to run.
To install pygame, you need to run pip install pygame-ce
